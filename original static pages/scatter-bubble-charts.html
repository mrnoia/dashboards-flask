üéØ Introduction to Scatter & Bubble Charts

Scatter and bubble charts are perfect for visualizing relationships between variables, identifying patterns, and showing distributions. They excel at correlation analysis and multi-dimensional data visualization.

ü´ß Scatter Charts
  • Shows correlation between variables
  • Identifies patterns and outliers
  • Multi-series support
  • Regression analysis ready
ü´ß Bubble Charts
  • Three-dimensional data points
  • Size represents third variable
  • Great for comparative analysis
  • Visual impact for data stories

üîß Basic Scatter Chart Implementation

HTML Structure:
<div class="container"> <div class="row"> <div class="col-md-12"> <div class="card"> <div class="card-header"> <h5>Height vs Weight Correlation</h5> </div> <div class="card-body"> <div id="basic-scatter-chart"></div> </div> </div> </div> </div> </div>
JavaScript Implementation:
// Basic Scatter Chart Configuration const options = { series: [{ name: 'Sample A', data: [[16.4, 5.4], [21.7, 2.0], [25.4, 3.0], [19.0, 2.0], [10.9, 1.0], [13.6, 3.7], [10.9, 1.0], [16.4, 5.4]] }, { name: 'Sample B', data: [[36.4, 13.4], [1.7, 11.0], [5.4, 8.0], [9.0, 5.0], [0.9, 3.0], [3.6, 4.7], [0.9, 0.0], [16.4, 5.4]] }], chart: { height: 350, type: 'scatter', zoom: { enabled: true, type: 'xy' } }, xaxis: { tickAmount: 10, labels: { formatter: function(val) { return parseFloat(val).toFixed(1) } } }, yaxis: { tickAmount: 7 }, colors: ['#3498db', '#e74c3c'] }; const chart = new ApexCharts(document.querySelector("#basic-scatter-chart"), options); chart.render();
Key Configuration Options:
  • type: 'scatter' - Sets the chart type to scatter
  • data: [[x, y], ...] - Array of coordinate pairs
  • zoom: { enabled: true, type: 'xy' } - Enable zoom functionality
  • tickAmount - Number of ticks on axes
  • formatter - Custom label formatting
Bootstrap Integration:

The scatter chart is wrapped in a Bootstrap card component for consistent styling and responsive layout. The card structure provides proper spacing and visual hierarchy.

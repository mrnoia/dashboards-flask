ğŸ“¦ Container Types

Fixed Container
Fixed width container that changes at responsive breakpoints
Fluid Container
100% width container, spanning the entire viewport
Responsive Container
100% wide until the specified breakpoint (md in this case)
Container Types:
<!-- Fixed Container --> <div class="container"> <!-- Content here --> </div> <!-- Fluid Container --> <div class="container-fluid"> <!-- Content here --> </div> <!-- Responsive Container --> <div class="container-sm">100% wide until small breakpoint</div> <div class="container-md">100% wide until medium breakpoint</div> <div class="container-lg">100% wide until large breakpoint</div> <div class="container-xl">100% wide until extra large breakpoint</div> <div class="container-xxl">100% wide until extra extra large breakpoint</div>
Container Breakpoints:
  • container-sm - 100% width until sm (576px)
  • container-md - 100% width until md (768px)
  • container-lg - 100% width until lg (992px)
  • container-xl - 100% width until xl (1200px)
  • container-xxl - 100% width until xxl (1400px)
Best Practices:
  • Use container for most layouts
  • Use container-fluid for full-width designs
  • Use responsive containers for specific breakpoint needs
  • Nest containers only when absolutely necessary

üéØ Introduction to Heatmaps & Treemaps

Heatmaps and treemaps are powerful visualization tools for showing intensity patterns and hierarchical relationships. They excel at displaying complex data in a compact, intuitive format.

üî• Heatmaps
  • Shows intensity through color
  • Matrix-style data display
  • Great for correlation analysis
  • Pattern identification
üó∫Ô∏è Treemaps
  • Hierarchical data visualization
  • Size represents value
  • Nested structure display
  • Space-efficient representation

üîß Basic Heatmap Implementation

HTML Structure:
<div class="container"> <div class="row"> <div class="col-md-12"> <div class="card"> <div class="card-header"> <h5>Activity Heatmap</h5> </div> <div class="card-body"> <div id="basic-heatmap-chart"></div> </div> </div> </div> </div> </div>
JavaScript Implementation:
// Basic Heatmap Configuration const options = { series: [{ name: 'Metric A', data: [[0, 0, 76], [0, 1, 45], [0, 2, 88], [0, 3, 19], [0, 4, 36]] }, { name: 'Metric B', data: [[1, 0, 44], [1, 1, 89], [1, 2, 55], [1, 3, 76], [1, 4, 12]] }, { name: 'Metric C', data: [[2, 0, 13], [2, 1, 28], [2, 2, 61], [2, 3, 83], [2, 4, 29]] }], chart: { height: 350, type: 'heatmap' }, dataLabels: { enabled: false }, colors: ["#3498db"], xaxis: { categories: ['Category A', 'Category B', 'Category C', 'Category D', 'Category E'] }, yaxis: { categories: ['Series 1', 'Series 2', 'Series 3'] } }; const chart = new ApexCharts(document.querySelector("#basic-heatmap-chart"), options); chart.render();
Key Configuration Options:
  • type: 'heatmap' - Sets the chart type to heatmap
  • data: [[x, y, value], ...] - Array of coordinate-value triples
  • dataLabels: { enabled: false } - Hide value labels on cells
  • colors - Color scale configuration
  • categories - Axis labels for x and y
Bootstrap Integration:

The heatmap is wrapped in a Bootstrap card component for consistent styling and responsive layout. The card structure provides proper spacing and visual hierarchy.

üé® Advanced Heatmap with Color Scale

Advanced Heatmap Configuration:
// Advanced Heatmap with Color Scale const advancedOptions = { series: [{ name: 'Jan', data: generateData(18, { min: 0, max: 90 }) }, { name: 'Feb', data: generateData(18, { min: 0, max: 90 }) }, { name: 'Mar', data: generateData(18, { min: 0, max: 90 }) }], chart: { height: 350, type: 'heatmap', toolbar: { show: false } }, dataLabels: { enabled: false }, colors: ["#3498db"], xaxis: { type: 'category', categories: ['10am', '11am', '12pm', '1pm', '2pm', '3pm', '4pm', '5pm', '6pm', '7pm', '8pm', '9pm', '10pm', '11pm', '12am', '1am', '2am', '3am'] }, yaxis: { categories: ['Jan', 'Feb', 'Mar'] }, grid: { borderColor: '#f1f1f1' } }; function generateData(count, yrange) { var i = 0; var series = []; while (i < count) { var x = 'w' + (i + 1).toString(); var y = Math.floor(Math.random() * (yrange.max - yrange.min + 1)) + yrange.min; series.push([x, y]); i++; } return series; } const advancedChart = new ApexCharts(document.querySelector("#advanced-heatmap-chart"), advancedOptions); advancedChart.render();

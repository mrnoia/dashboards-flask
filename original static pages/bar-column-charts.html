üìä Bar & Column Charts Tutorial

Master horizontal and vertical bar charts with ApexCharts and Bootstrap 5

What You'll Learn:
  • Basic bar chart configuration
  • Column chart variations
  • Stacked and grouped bars
  • Custom styling and animations
  • Responsive design integration
Prerequisites:
  • Basic HTML knowledge
  • JavaScript fundamentals
  • Bootstrap 5 basics
  • ApexCharts installation

üéØ Introduction to Bar & Column Charts

Bar and column charts are among the most common and effective ways to visualize categorical data. They excel at comparing values across different categories and showing trends over time.

üìä Bar Charts (Horizontal)
  • Best for comparing categories
  • Long category names work well
  • Easy to read values
  • Space-efficient for many categories
üìà Column Charts (Vertical)
  • Best for time series data
  • Shows trends clearly
  • Familiar to most users
  • Good for positive/negative values

üîß Basic Bar Chart Implementation

HTML Structure:
<div class="container"> <div class="row"> <div class="col-md-12"> <div class="card"> <div class="card-header"> <h5>Monthly Sales Data</h5> </div> <div class="card-body"> <div id="basic-bar-chart"></div> </div> </div> </div> </div> </div>
JavaScript Implementation:
// Basic Bar Chart Configuration const options = { series: [{ name: 'Sales', data: [44, 55, 41, 67, 22, 43, 21, 33, 45, 31, 47, 28] }], chart: { type: 'bar', height: 350, toolbar: { show: false } }, plotOptions: { bar: { horizontal: true, borderRadius: 4, dataLabels: { position: 'top', } } }, dataLabels: { enabled: true, offsetX: -6, style: { fontSize: '12px', colors: ['#fff'] } }, xaxis: { categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'], }, colors: ['#3498db'] }; const chart = new ApexCharts(document.querySelector("#basic-bar-chart"), options); chart.render();
Key Configuration Options:
  • type: 'bar' - Sets the chart type to bar
  • horizontal: true - Creates horizontal bars
  • borderRadius: 4 - Adds rounded corners to bars
  • dataLabels - Shows values on top of bars
  • categories - Defines x-axis labels
  • colors - Sets bar colors
Bootstrap Integration:

The chart is wrapped in Bootstrap cards for consistent styling and responsive layout. The card structure provides proper spacing and visual hierarchy.

üìà Column Chart with Multiple Series

Multi-Series Column Chart:
// Multi-Series Column Chart const columnOptions = { series: [{ name: 'Product A', data: [44, 55, 41, 67, 22, 43] }, { name: 'Product B', data: [13, 23, 20, 8, 13, 27] }, { name: 'Product C', data: [11, 17, 15, 15, 21, 14] }], chart: { type: 'bar', height: 350, stacked: false, toolbar: { show: false } }, plotOptions: { bar: { horizontal: false, columnWidth: '55%', endingShape: 'rounded' }, }, dataLabels: { enabled: false }, stroke: { show: true, width: 2, colors: ['transparent'] }, xaxis: { categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'], }, yaxis: { title: { text: 'Sales ($)' } }, fill: { opacity: 1 }, tooltip: { y: { formatter: function (val) { return "$" + val + " thousands" } } }, colors: ['#3498db', '#2ecc71', '#e74c3c'] }; const columnChart = new ApexCharts(document.querySelector("#column-chart"), columnOptions); columnChart.render();

üìö Stacked Bar Chart

Stacked Bar Chart Configuration:
// Stacked Bar Chart const stackedOptions = { series: [{ name: 'Q1', data: [44, 55, 41, 67, 22, 43] }, { name: 'Q2', data: [13, 23, 20, 8, 13, 27] }, { name: 'Q3', data: [11, 17, 15, 15, 21, 14] }], chart: { type: 'bar', height: 350, stacked: true, toolbar: { show: false } }, plotOptions: { bar: { horizontal: true, dataLabels: { total: { enabled: true, offsetX: 0, style: { fontSize: '13px', fontWeight: 900 } } } }, }, stroke: { width: 1, colors: ['#fff'] }, xaxis: { categories: ['Product A', 'Product B', 'Product C', 'Product D', 'Product E', 'Product F'], }, yaxis: { title: { text: undefined }, }, tooltip: { y: { formatter: function (val) { return val + "K" } } }, fill: { opacity: 1 }, legend: { position: 'top', horizontalAlign: 'left', offsetX: 40 }, colors: ['#3498db', '#2ecc71', '#e74c3c'] }; const stackedChart = new ApexCharts(document.querySelector("#stacked-chart"), stackedOptions); stackedChart.render();

HTML Structure

<!-- Basic Card -->
<div class="card">
  <img src="image.jpg" class="card-img-top" alt="Card image">
  <div class="card-body">
    <h5 class="card-title">Card Title</h5>
    <p class="card-text">Card content goes here.</p>
    <a href="#" class="btn btn-primary">Action</a>
  </div>
</div>

<!-- Modal Trigger -->
<button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
  Open Modal
</button>

<!-- Modal Structure -->
<div class="modal fade" id="myModal" tabindex="-1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">Modal Title</h5>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>
      <div class="modal-body">
        <p>Modal content goes here.</p>
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
        <button type="button" class="btn btn-primary">Save changes</button>
      </div>
    </div>
  </div>
</div>

JavaScript for Modals

// Initialize modal with JavaScript
const myModal = new bootstrap.Modal(document.getElementById('myModal'));

// Show modal
myModal.show();

// Hide modal
myModal.hide();

// Listen to modal events
const modalElement = document.getElementById('myModal');
modalElement.addEventListener('show.bs.modal', function (event) {
  // Do something when modal is about to be shown
});

modalElement.addEventListener('hidden.bs.modal', function (event) {
  // Do something when modal is hidden
});

Bootstrap Cards

Cards are flexible content containers built with minimal markup and styling options. They feature:

Card Structure

  • .card: Main container wrapper
  • .card-body: Main content area
  • .card-title: Card heading
  • .card-text: Main text content
  • .card-header/.card-footer: Optional header/footer sections
  • .card-img-top/.card-img-bottom: Top/bottom images

Card Variations

  • Basic cards: Simple content containers
  • Image cards: Cards with images
  • List group cards: Cards with list content
  • Header/footer cards: Cards with structural sections
  • Text alignment: Use text utilities for alignment

Bootstrap Modals

Modals are flexible dialog prompts that require user interaction before continuing:

Modal Components

  • .modal: Root modal container
  • .modal-dialog: Modal positioning and sizing
  • .modal-content: Modal content wrapper
  • .modal-header: Header with title and close button
  • .modal-body: Main content area
  • .modal-footer: Footer with action buttons

Modal Triggers

  • Data attributes: Use data-bs-toggle="modal" and data-bs-target
  • JavaScript: Use Bootstrap.Modal constructor
  • Events: show.bs.modal, shown.bs.modal, hide.bs.modal, hidden.bs.modal

Best Practices

  • Use semantic HTML5 tags within cards
  • Ensure modals are accessible with proper ARIA attributes
  • Test responsive behavior on different screen sizes
  • Use appropriate modal sizes (modal-sm, modal-lg, modal-xl)
  • Consider backdrop options for user experience

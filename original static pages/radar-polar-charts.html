üéØ Introduction to Radar & Polar Charts

Radar and polar charts are perfect for comparing multiple variables across different dimensions. They excel at showing performance metrics, skill assessments, and multi-dimensional comparisons.

üîß Basic Radar Chart Implementation

HTML Structure:
<div class="container"> <div class="row"> <div class="col-md-12"> <div class="card"> <div class="card-header"> <h5>Skill Assessment</h5> </div> <div class="card-body"> <div id="basic-radar-chart"></div> </div> </div> </div> </div> </div>
JavaScript Implementation:
// Basic Radar Chart Configuration const options = { series: [{ name: 'Series 1', data: [80, 50, 30, 40, 100, 20] }], chart: { height: 350, type: 'radar' }, xaxis: { categories: ['Math', 'Science', 'History', 'Geography', 'English', 'Art'] }, yaxis: { show: true, labels: { show: true, formatter: function(val) { return val + '%' } } }, colors: ['#3498db'] }; const chart = new ApexCharts(document.querySelector("#basic-radar-chart"), options); chart.render();
Key Configuration Options:
  • type: 'radar' - Sets the chart type to radar
  • data: [values] - Array of values for each dimension
  • categories - Labels for each axis
  • yaxis: { show: true } - Show/hide axis values
  • formatter - Custom label formatting
Bootstrap Integration:

The radar chart is wrapped in a Bootstrap card component for consistent styling and responsive layout. The card structure provides proper spacing and visual hierarchy.

üìä Multi-Series Radar Chart

Multi-Series Radar Chart Configuration:
// Multi-Series Radar Chart const multiOptions = { series: [{ name: 'Team A', data: [44, 55, 41, 17, 15] }, { name: 'Team B', data: [13, 23, 20, 8, 13] }, { name: 'Team C', data: [11, 17, 15, 15, 21] }], chart: { height: 350, type: 'radar', dropShadow: { enabled: true, blur: 1, left: 1, top: 1 } }, stroke: { width: 2 }, fill: { opacity: 0.1 }, markers: { size: 0 }, xaxis: { categories: ['2011', '2012', '2013', '2014', '2015'] }, yaxis: { show: true, labels: { show: true, formatter: function(val) { return val + 'k' } } }, colors: ['#3498db', '#2ecc71', '#e74c3c'] }; const multiChart = new ApexCharts(document.querySelector("#multi-radar-chart"), multiOptions); multiChart.render();

üåê Polar Area Chart

Polar Area Chart Configuration:
// Polar Area Chart const polarOptions = { series: [42, 47, 52, 38, 25], chart: { type: 'polarArea', height: 350 }, labels: ['Rose A', 'Rose B', 'Rose C', 'Rose D', 'Rose E'], fill: { opacity: 0.8 }, stroke: { width: 0, colors: undefined }, yaxis: { show: false }, legend: { position: 'bottom' }, plotOptions: { polarArea: { rings: { strokeWidth: 0 }, spokes: { strokeWidth: 0 } } }, colors: ['#3498db', '#2ecc71', '#e74c3c', '#f39c12', '#9b59b6'] }; const polarChart = new ApexCharts(document.querySelector("#polar-chart"), polarOptions); polarChart.render();

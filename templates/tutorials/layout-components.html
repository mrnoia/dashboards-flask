Bootstrap Layout System

Bootstrap provides a flexible, mobile-first grid system with responsive breakpoints and utilities.

Containers

  • .container: Fixed-width container that changes at breakpoints
  • .container-fluid: Full-width container spanning entire viewport
  • .container-{breakpoint}: 100% wide until specified breakpoint
  • Containers provide padding and center content horizontally

Grid System

  • 12-column system: Divide content into up to 12 columns
  • .row: Horizontal group of columns
  • .col: Equal-width columns
  • .col-{breakpoint}-{number}: Specific column widths at breakpoints
  • Gutters: Space between columns (controlled by .g-* classes)

Breakpoints

  • xs: 0px and up (no prefix needed)
  • sm: 576px and up
  • md: 768px and up
  • lg: 992px and up
  • xl: 1200px and up
  • xxl: 1400px and up

Grid Options

  • Auto-layout columns: Equal width without specifying numbers
  • Column wrapping: Columns automatically wrap to new lines
  • Column breaks: Force columns to new lines with .w-100
  • Ordering: Change column order with .order-* classes
  • Offsetting: Push columns with .offset-* classes

Flexbox Utilities

Display Utilities

  • .d-flex: Create flex container
  • .d-inline-flex: Create inline flex container
  • Responsive: .d-{breakpoint}-flex for responsive behavior

Direction

  • .flex-row: Left to right (default)
  • .flex-row-reverse: Right to left
  • .flex-column: Top to bottom
  • .flex-column-reverse: Bottom to top

Justify Content

  • .justify-content-start: Items at start (default)
  • .justify-content-end: Items at end
  • .justify-content-center: Items centered
  • .justify-content-between: Space between items
  • .justify-content-around: Space around items

Align Items

  • .align-items-start: Align to start
  • .align-items-end: Align to end
  • .align-items-center: Align to center
  • .align-items-baseline: Align to baseline
  • .align-items-stretch: Stretch to fill (default)

Best Practices

  • Always use containers for proper spacing and centering
  • Design mobile-first (start with small screens)
  • Use semantic HTML5 elements within layouts
  • Test layouts at all breakpoints
  • Use flexbox utilities for component-level layouts
  • Combine grid and flexbox for complex layouts
  • Consider accessibility when creating layouts

Common Patterns

  • Hero sections: Full-width featured content areas
  • Card grids: Responsive card layouts with equal heights
  • Sidebar layouts: Main content with side navigation
  • Masonry grids: Pinterest-style layouts
  • Nested grids: Complex multi-level layouts

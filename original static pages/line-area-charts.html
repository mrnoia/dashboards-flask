üéØ Introduction to Line & Area Charts

Line and area charts are perfect for visualizing trends over time, showing continuous data, and displaying relationships between variables. They're essential for time series analysis and trend monitoring.

üìà Line Charts
  • Best for time series data
  • Clear trend visualization
  • Multiple series support
  • Precise value reading
üìä Area Charts
  • Emphasizes volume/quantity
  • Visual impact for trends
  • Stacked area support
  • Gradient fills possible

üîß Basic Line Chart Implementation

HTML Structure:
<div class="container"> <div class="row"> <div class="col-md-12"> <div class="card"> <div class="card-header"> <h5>Monthly Revenue Trend</h5> </div> <div class="card-body"> <div id="basic-line-chart"></div> </div> </div> </div> </div> </div>
JavaScript Implementation:
// Basic Line Chart Configuration const options = { series: [{ name: 'Revenue', data: [30, 40, 35, 50, 49, 60, 70, 91, 125] }], chart: { height: 350, type: 'line', zoom: { enabled: false } }, dataLabels: { enabled: false }, stroke: { curve: 'straight' }, title: { text: 'Product Trends by Month', align: 'left' }, grid: { row: { colors: ['#f3f3f3', 'transparent'], opacity: 0.5 }, }, xaxis: { categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep'], }, colors: ['#3498db'] }; const chart = new ApexCharts(document.querySelector("#basic-line-chart"), options); chart.render();
Key Configuration Options:
  • type: 'line' - Sets the chart type to line
  • stroke: { curve: 'straight' } - Line style (straight, smooth, stepline)
  • zoom: { enabled: false } - Disable zoom functionality
  • dataLabels: { enabled: false } - Hide value labels on points
  • grid - Background grid configuration
  • colors - Line color configuration
Bootstrap Integration:

The line chart is wrapped in a Bootstrap card component for consistent styling and responsive layout. The card structure provides proper spacing and visual hierarchy.

üìä Area Chart with Gradient Fill

Area Chart with Gradient:
// Area Chart with Gradient Fill const areaOptions = { series: [{ name: 'Sales', data: [31, 40, 28, 51, 42, 109, 100] }, { name: 'Revenue', data: [11, 32, 45, 32, 34, 52, 41] }], chart: { height: 350, type: 'area', toolbar: { show: false } }, dataLabels: { enabled: false }, stroke: { curve: 'smooth' }, xaxis: { categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul'], }, tooltip: { x: { format: 'dd/MM/yy' }, }, fill: { type: 'gradient', gradient: { shadeIntensity: 1, opacityFrom: 0.7, opacityTo: 0.9, stops: [0, 90, 100] } }, colors: ['#3498db', '#2ecc71'] }; const areaChart = new ApexCharts(document.querySelector("#area-chart"), areaOptions); areaChart.render();

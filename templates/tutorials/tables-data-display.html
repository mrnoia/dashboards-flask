Basic Table Structure

<table class="table">
  <thead>
    <tr>
      <th scope="col">Header 1</th>
      <th scope="col">Header 2</th>
      <th scope="col">Header 3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td>

      <td>Data 3</td>
    </tr>

Bootstrap Tables

Bootstrap tables provide a flexible and responsive way to display tabular data with minimal markup.

Table Structure

  • .table: Base class for all tables
  • .table-striped: Adds zebra-striping to rows
  • .table-bordered: Adds borders on all sides
  • .table-borderless: Removes all borders
  • .table-hover: Enables hover state on rows
  • .table-sm: Makes table more compact

Responsive Tables

  • Wrap tables in .table-responsive for horizontal scrolling
  • Use .table-responsive-sm/md/lg/xl for breakpoint-specific responsiveness
  • Essential for mobile devices and small screens

Table Styling

  • .table-dark: Dark theme for table
  • .table-light: Light theme for header
  • Contextual classes: Color rows and cells (success, danger, warning, info, etc.)
  • .thead-dark/light: Style table headers

Data Display Components

List Groups

  • Flexible component for displaying lists of content
  • Supports badges, actions, and custom content
  • Can be used as navigation elements
  • .list-group-flush: Removes outer borders

Description Lists

  • Semantic way to display key-value pairs
  • Use dl, dt, and dd elements
  • Grid system integration with .row classes
  • Ideal for metadata and specifications

Badges and Labels

  • .badge: Small indicators and labels
  • Contextual colors for status indication
  • .rounded-pill: Pill-shaped badges
  • Perfect for status indicators and counts

Best Practices

  • Always use .table-responsive for mobile compatibility
  • Use semantic HTML5 elements (thead, tbody, th, td)
  • Add proper scope attributes to table headers
  • Use contextual colors to convey meaning, not just decoration
  • Consider pagination for large datasets
  • Test tables on various screen sizes

Advanced Features

  • Combine with JavaScript for sorting and filtering
  • Use with DataTables plugin for advanced functionality
  • Implement pagination for large datasets
  • Add search functionality for better UX
  • Consider server-side processing for very large tables

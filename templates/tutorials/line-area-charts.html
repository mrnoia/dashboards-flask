Basic Line Chart

// Basic Line Chart Configuration
const options = {
  series: [{
    name: "Sales",
    data: [30, 40, 35, 50, 49, 60, 70, 91, 125]
  }],
  chart: {
    height: 350,
    type: 'line',
    zoom: {
      enabled: false
    }
  },
  dataLabels: {
    enabled: false
  },
  stroke: {
    curve: 'straight'
  },
  title: {
    text: 'Monthly Sales',
    align: 'left'
  },
  grid: {
    row: {
      colors: ['#f3f3f3', 'transparent'],
      opacity: 0.5
    }
  },
  xaxis: {
    categories: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep'],
  }
};

const chart = new ApexCharts(document.querySelector("#basicLineChart"), options);
chart.render();

Line & Area Charts Overview

Line and area charts are perfect for showing trends over time or comparing multiple data series.

Line Charts

  • Purpose: Show trends and changes over time
  • Data points: Connected with lines
  • Best for: Time series, continuous data
  • Variations: Smooth curves, straight lines, stepped

Area Charts

  • Purpose: Show magnitude of change over time
  • Visual impact: Filled area under the line
  • Best for: Volume, cumulative data, part-to-whole
  • Variations: Stacked, overlapping, gradient fills

Chart Configuration Options

Basic Settings

  • chart.type: 'line' or 'area'
  • chart.height: Chart height in pixels
  • chart.zoom: Enable zoom functionality
  • chart.toolbar: Show/hide toolbar

Series Configuration

  • series.name: Series name for legend
  • series.data: Array of data points
  • Multi-series: Array of series objects
  • Time series: [timestamp, value] pairs

Styling Options

  • stroke.curve: 'straight', 'smooth', 'stepline'
  • stroke.width: Line thickness
  • colors: Array of colors for series
  • markers: Data point markers

Fill Options

  • fill.type: 'solid', 'gradient', 'pattern'
  • fill.opacity: Fill transparency
  • gradient: Gradient configuration
  • stacked: Stack multiple series

Data Visualization Best Practices

When to Use Line Charts

  • Time series data
  • Showing trends over time
  • Comparing multiple categories
  • Continuous data progression

When to Use Area Charts

  • Showing volume or magnitude
  • Part-to-whole relationships
  • Cumulative totals
  • Visual impact emphasis

Design Guidelines

  • Limit to 5-7 series maximum
  • Use contrasting colors for series
  • Include clear axis labels
  • Add tooltips for detailed values
  • Consider data density and readability

Advanced Features

Interactivity

  • Zoom: Zoom into specific time ranges
  • Pan: Navigate through zoomed data
  • Selection: Select data ranges
  • Hover: Detailed tooltips on hover

Time Series Features

  • Datetime x-axis: Proper time formatting
  • Time zones: Handle different time zones
  • Missing data: Handle gaps in data
  • Real-time updates: Dynamic data updates

Performance Considerations

  • Limit data points for performance
  • Use sampling for large datasets
  • Consider server-side aggregation
  • Optimize for mobile devices

Common Use Cases

  • Financial data: Stock prices, revenue trends
  • Analytics: Website traffic, user metrics
  • Sales data: Revenue, units sold over time
  • Weather data: Temperature, precipitation trends
  • Performance metrics: System performance over time

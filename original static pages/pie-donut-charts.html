üéØ Introduction to Pie & Donut Charts

Pie and donut charts are perfect for showing proportions and percentages of a whole. They excel at displaying composition data and making it easy to understand relative sizes at a glance.

ü•ß Pie Charts
  • Classic proportional visualization
  • Easy to understand at a glance
  • Best for 5-7 categories max
  • Clear percentage representation
üç© Donut Charts
  • Modern alternative to pie charts
  • Center space for additional info
  • Less visual weight than pie charts
  • Great for dashboards and reports

üîß Basic Pie Chart Implementation

HTML Structure:
<div class="container"> <div class="row"> <div class="col-md-12"> <div class="card"> <div class="card-header"> <h5>Market Share Distribution</h5> </div> <div class="card-body"> <div id="basic-pie-chart"></div> </div> </div> </div> </div> </div>
JavaScript Implementation:
// Basic Pie Chart Configuration const options = { series: [44, 55, 13, 43, 22], chart: { type: 'pie', height: 350 }, labels: ['Team A', 'Team B', 'Team C', 'Team D', 'Team E'], responsive: [{ breakpoint: 480, options: { chart: { width: 200 }, legend: { position: 'bottom' } } }], colors: ['#3498db', '#2ecc71', '#e74c3c', '#f39c12', '#9b59b6'] }; const chart = new ApexCharts(document.querySelector("#basic-pie-chart"), options); chart.render();
Key Configuration Options:
  • type: 'pie' - Sets the chart type to pie
  • series - Array of values for each segment
  • labels - Labels for each segment
  • responsive - Responsive configuration for different screen sizes
  • colors - Custom colors for each segment
Bootstrap Integration:

The pie chart is wrapped in a Bootstrap card component for consistent styling and responsive layout. The card structure provides proper spacing and visual hierarchy.

üç© Donut Chart with Custom Labels

Donut Chart with Custom Labels:
// Donut Chart with Custom Labels const donutOptions = { series: [44, 55, 41, 17, 15], chart: { type: 'donut', height: 350 }, plotOptions: { pie: { donut: { size: '70%', labels: { show: true, total: { show: true, label: 'Total', formatter: function (w) { return w.globals.seriesTotals.reduce((a, b) => { return a + b }, 0) } } } } } }, labels: ['Product A', 'Product B', 'Product C', 'Product D', 'Product E'], dataLabels: { enabled: true, formatter: function (val, opts) { return opts.w.globals.labels[opts.seriesIndex] + ': ' + val + '%' } }, responsive: [{ breakpoint: 480, options: { chart: { width: 200 }, legend: { position: 'bottom' } } }], colors: ['#3498db', '#2ecc71', '#e74c3c', '#f39c12', '#9b59b6'] }; const donutChart = new ApexCharts(document.querySelector("#donut-chart"), donutOptions); donutChart.render();

üéØ Introduction to Cards & Modals

Cards and modals are essential UI components for organizing content and creating interactive user experiences. Cards provide flexible content containers, while modals offer dialog overlays for user interactions.

üé¥ Card Components

  • Image and text integration
  • Responsive grid layouts
ü™ü Modal Dialogs

  • Multiple trigger options

Key Card Classes:
  • card - Base card container
  • card-img-top - Top image styling
  • card-body - Main content area
  • card-title - Title styling
  • card-text - Text content styling
  • card-header - Header section
  • card-footer - Footer section
Card Variations:
  • card-img-overlay - Text overlay on image
  • card-img-bottom - Bottom image
  • card-subtitle - Subtitle styling
  • card-link - Styled links
  • card-group - Equal height cards
  • card-deck - Card grid layout

Navbar Structure

<!-- Basic Navbar -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Brand</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav me-auto">
        <li class="nav-item">
          <a class="nav-link active" href="#">Home</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Features</a>
        </li>
        <li class="nav-item dropdown">
          <a class="nav-link dropdown-toggle" href="#" id="navbarDropdown" role="button" data-bs-toggle="dropdown">
            Dropdown
          </a>
          <ul class="dropdown-menu">
            <li><a class="dropdown-item" href="#">Action</a></li>
            <li><a class="dropdown-item" href="#">Another action</a></li>
          </ul>
        </li>
      </ul>
    </div>
  </div>
</nav>

Bootstrap Navigation Components

Bootstrap provides a comprehensive set of navigation components for building intuitive user interfaces.

Navbar

  • .navbar: Base class for navigation bar
  • .navbar-expand-{breakpoint}: Responsive collapse behavior
  • .navbar-brand: Company/product branding
  • .navbar-nav: Full-height navigation
  • .nav-link: Individual navigation links
  • .navbar-toggler: Mobile menu toggle button
  • .navbar-collapse: Collapsible content wrapper

Navbar Color Schemes

  • .navbar-light: Light theme for light backgrounds
  • .navbar-dark: Dark theme for dark backgrounds
  • Background utilities: .bg-light, .bg-dark, .bg-primary, etc.
  • Placement: .fixed-top, .fixed-bottom, .sticky-top

Navbar Content

  • Navigation links: Primary navigation items
  • Dropdowns: Multi-level navigation menus
  • Forms: Search bars and login forms
  • Text: Additional text content
  • Buttons: Action buttons and CTAs

Breadcrumb Navigation

  • Shows hierarchical navigation path
  • Indicates current page location
  • Provides easy navigation back to parent pages
  • Uses semantic HTML5 elements
  • Accessible with proper ARIA attributes

Pagination

  • .pagination: Container for pagination controls
  • .page-item: Individual pagination item
  • .page-link: Clickable pagination link
  • Sizes: .pagination-lg, .pagination-sm
  • States: .active, .disabled
  • Alignment: Use flexbox utilities

Tabs and Pills

  • .nav-tabs: Tab-style navigation
  • .nav-pills: Pill-style navigation
  • .nav-link: Clickable navigation item
  • .tab-content: Content container
  • .tab-pane: Individual tab content
  • Vertical: .flex-column for vertical orientation

Navigation Best Practices

  • Keep navigation simple and intuitive
  • Use clear, descriptive labels
  • Ensure mobile responsiveness
  • Provide visual feedback for active states
  • Use semantic HTML5 elements
  • Test keyboard accessibility
  • Consider touch-friendly targets on mobile

JavaScript Requirements

  • Dropdowns require Bootstrap dropdown plugin
  • Tabs require Bootstrap tab plugin
  • Collapse functionality for mobile menus
  • All components work with Bootstrap's JavaScript
  • Can be initialized with data attributes or JavaScript

Accessibility Considerations

  • Use proper ARIA attributes
  • Ensure keyboard navigation support
  • Provide sufficient color contrast
  • Use semantic HTML5 elements
  • Test with screen readers
  • Provide focus indicators
